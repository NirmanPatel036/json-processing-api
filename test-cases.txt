-------------------------------------------------------------------
TEST CASE 1: Filter Data (Success)
-------------------------------------------------------------------
Description: Filter a dataset to find items with category "electronics".
Expected Status: 200 OK
Body:
{
  "operation": "filter",
  "dataset": [
    {"id": 1, "name": "Laptop", "category": "electronics", "price": 1000},
    {"id": 2, "name": "Banana", "category": "food", "price": 2},
    {"id": 3, "name": "Mouse", "category": "electronics", "price": 50}
  ],
  "criteria": {
    "category": "electronics"
  }
}

-------------------------------------------------------------------
TEST CASE 2: Sort Data by Number (Success)
-------------------------------------------------------------------
Description: Sort a dataset by "price" in ascending order.
Expected Status: 200 OK
Body:
{

    {"item": "A", "price": 100},
    {"item": "B", "price": 20},

TEST CASE 7: Filter - No Matches (Success - Edge Case)
-------------------------------------------------------------------
Description: Filter with criteria that matches nothing. Should return empty array.
Expected Status: 200 OK
Body:
{
  "operation": "filter",
  "dataset": [
    {"id": 1, "status": "active"},
    {"id": 2, "status": "active"}
  ],
  "criteria": {
    "status": "archived"
  }
}

-------------------------------------------------------------------
TEST CASE 8: Error - Invalid Operation (Logic Error)
-------------------------------------------------------------------
Description: Send an operation name that is not defined in the code.
Expected Status: 400 Bad Request
Body:
{
  "operation": "delete_database", 
  "dataset": [{"id": 1}]
}

-------------------------------------------------------------------
TEST CASE 9: Error - Missing Required Field (Validation Error)
-------------------------------------------------------------------
Description: Omit the "dataset" field, which is required by JSON Schema.
Expected Status: 400 Bad Request
Body:
{
  "operation": "filter",
  "criteria": {"id": 1}
}

-------------------------------------------------------------------
TEST CASE 10: Error - Wrong Data Type (Validation Error)
-------------------------------------------------------------------
Description: Send "dataset" as a string instead of an array.
Expected Status: 400 Bad Request
Body:
{
  "operation": "sort",
  "dataset": "This should be an array, not a string",
  "sortBy": "id"
}
